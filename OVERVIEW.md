# ReceiptLog

## Background
Receipts are easy to lose, damage, or forget, yet they are critical for accurate expense tracking and reporting. Even with powerful accounting tools, users still rely on manual data entry—wasting time and introducing errors.

Meanwhile, people already use WhatsApp and Telegram every day to share photos and documents, including receipts. ReceiptLog takes advantage of this habit by turning simple receipt uploads into structured expense data automatically. With AI-powered scanning and flexible exports, ReceiptLog makes expense tracking fast, accurate, and effortless—without changing how people already work.

## Key Features
- AI-powered receipt scanning and data extraction  
- WhatsApp and Telegram bot for receipt uploads  
- Expense entry management and reporting dashboard (Web / Android)  
- Customizable expense report formats with CSV/XLS export  
- Public APIs for modular integration with existing systems  

## Key Technologies
- React (Web UI)  
- Vercel (Deployment & hosting)  
- Gemini API (AI/OCR and data extraction)  

## Competitive Advantages
- **Fast**: Upload receipts directly from chat apps without context switching  
- **Robust**: AI-driven extraction reduces manual errors  
- **Easy**: Minimal setup, familiar interfaces, and flexible exports
